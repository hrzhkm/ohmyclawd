⚡ OhMyClawd

Claude Code usage monitor for ESP32-2432S028R (CYD 2.8")

Flash Firmware

Flash ohmyclawd directly from your browser. No PlatformIO or esptool needed.

  1. Connect your CYD via USB
  2. Click the button below
  3. Select the serial port
  4. Wait for flash to complete
Your browser doesn't support Web Serial. Use Chrome, Edge, or Opera. Web Serial requires HTTPS.

Requires Chrome, Edge, or Opera. First flash erases the device.

USB driver needed? CH340 · CP210x

Linux users: add yourself to the dialout group
